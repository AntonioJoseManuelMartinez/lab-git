# lab-git

Repository for Lab 3 of the course Cloud, HPC & Bio (Master in Bioinformatics, University of Murcia). Includes all Git/GitHub tasks: local repository creation, commits, branching, merging, conflict resolution, and collaborative work using pull requests. Also contains the file-cut.sh script, the submit_lab_machine-name-login.sh SLURM script, the job output file, and the final report.
